# [공유] Claude Code 사용량을 트레이 아이콘으로 실시간 확인하기

> **[다운로드 (최신 버전)](https://github.com/Sayh0/ullmananamazzi/releases/latest)** — zip 다운로드 → 압축 해제 → `ullmananamazzi.exe` 실행

**G_One E4U**

&nbsp;

## 1. 문제 인식 — "지금 사용량이 얼마나 됐지?"

**Claude Code를 쓰다 보면 어느 순간 갑자기 응답이 막힙니다.**

**5시간 세션 한도가 언제 찼는지, 주간 한도는 얼마나 남았는지 Claude Code 자체에서는 확인할 방법이 없습니다.**

**매번 claude.ai 웹을 열어 확인하는 것이 번거로웠습니다.**

&nbsp;

## 2. 방법 탐색 — 선택지 비교

| 방법 | 장점 | 단점 |
|------|------|------|
| claude.ai 웹 직접 확인 | 공식, 정확 | 매번 브라우저 열어야 함 |
| Claude Code 내 명령어 | 없음 | 사용량 조회 기능 자체가 없음 |
| **Windows 트레이 위젯** | **항상 눈에 보임, 자동 갱신** | **비공식 API 사용** |

**→ 개발 중 화면 전환 없이 확인할 수 있도록 트레이 위젯을 직접 만들었습니다.**

&nbsp;

## 3. 주요 기능

![트레이 아이콘](capture/capture1.png) ![Simple 모드](capture/capture2.png) ![Detail 모드](capture/capture3.png)

**🟠 트레이 아이콘 — 사용률 실시간 표시**

| 상태 | 아이콘 |
|------|--------|
| 정상 | 주황 배경 + 숫자 (예: `25`) |
| 소진 (100%) | 빨간 배경 + `!!` |
| 데이터 없음 | 회색 배경 + `--` |

**화면을 보지 않아도 트레이만 보면 현재 사용률을 즉시 파악할 수 있습니다.**

&nbsp;

**🗂️ 위젯 팝업 — 상세 사용량 확인**

**트레이 아이콘 클릭 시 팝업이 열리며 항목별 프로그레스 바와 리셋까지 남은 시간을 표시합니다. 헤더에는 Clawd 스크래치 애니메이션이 표시됩니다.**

| 항목 | 내용 |
|------|------|
| 5H SESSION | 5시간 세션 사용량 + 리셋까지 남은 시간 |
| WEEKLY | 주간 사용량 + 리셋까지 남은 시간 |

&nbsp;

**🔐 토큰 자동 갱신**

**Claude Code의 OAuth 토큰을 재활용하므로 별도 로그인이 없습니다.**

**토큰 만료 시 자동 갱신하며, Claude Code CLI 재실행이 필요한 경우 위젯이 자동으로 열려 안내합니다.**

&nbsp;

## 4. 세팅 — 5분 이내 완료

> **필수 전제조건**: [Claude Code CLI](https://claude.ai/download)가 설치되어 있고 로그인된 상태여야 합니다. 이 위젯은 Claude Code가 저장한 OAuth 토큰을 재활용합니다.

**Step 1. 실행 파일 다운로드**

```
https://github.com/Sayh0/ullmananamazzi/releases
```

**최신 zip을 다운로드하고 압축을 풀어 원하는 위치에 복사합니다.**

&nbsp;

**Step 2. 실행**

**폴더 안의 `ullmananamazzi.exe` 실행 시 트레이 아이콘이 나타납니다.**

**Claude Code가 설치되어 있고 로그인된 상태라면 별도 설정 없이 바로 동작합니다.**

> **첫 실행 시**: Claude Code CLI 로그인이 안 된 상태이면 위젯이 자동으로 열려 안내 메시지를 표시합니다. 터미널에서 `claude` 실행 후 `/login`으로 로그인하세요.
